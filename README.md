# 🏢 Dormitory Management System API

Backend system built with **ASP.NET** Core and **SQL Server** to manage dormitory operations such as students, rooms, floors, payments, guests, and invitations.

---

## 📌 What this project does

This API centralizes dormitory management by allowing administrators to organize the building structure and monitor operations, while students can manage their own payments, guests, and invitations through secured access.

---

## ✨ Main Features

- 🔐 JWT Authentication & Role-Based Authorization (Admin / Student)  
- 👨‍🎓 Student management  
- 🏢 Floors and rooms management  
- 🛏️ Room allocation and availability tracking  
- 💳 Payments management and history tracking  
- 👥 Guest registration and monitoring  
- 📩 Invitation system between students  
- 🔒 Secure protected endpoints  

---

## 🛠️ Tech Stack

- ASP.NET Core (.NET)
- Microsoft SQL Server
- Entity Framework Core
- JWT Authentication
- RESTful API architecture
